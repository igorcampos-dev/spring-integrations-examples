# spring-kafka-example

---

## Overview

Basic Java Spring (Java 21) project showing simple integration with Apache Kafka.  
Includes a Kafka producer and consumer with minimal setup for local development.

---

## Tech stack

- Spring Boot
- Java 21
- Kafka (Docker)
